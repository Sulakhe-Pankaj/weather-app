# Weather App (Flask) Live

This Flask-based Weather App is **LIVE** and running on Render.

## Live Demo
**App URL:** `https://weather-app-z0ax.onrender.com`

## What it shows
- Weather data: **Temperature, Description, Min/Max Temp, Humidity, Pressure, Wind Speed, Weather Icon**
- Error message displayed for invalid city input
- Full screen background video
- Interactive animated UI inside the weather card and search bar

## Tech Used
- Flask (Backend)
- OpenWeather API (Weather Data)
- Requests (API Calls)
- Gunicorn (Server for deployment)
- CSS Animations + Video background
- Bootstrap Icons (UI Icons)
